BRESNIK

PEPSI NEXT SUPERBOWL

Free Pepsi Next Coupon support for a Pepsi Next Superbowl commercial. Allowed anyone to use a simple form to request a free PepsiNext 1 liter bottle, valid for only 10 minutes. Set the benchmark at 10 million requests in 10 seconds (estimated based on a percentage of the number of Superbowl watchers from the previous year). Used Nodejs to provide asynchronous persistence to the database so each request could return immediately. Used benchmark to establish the necessary amount of infrastructure required. Day of the Superbowl it processed 19 million requests the first minute, 32 million in the remaining minutes with 0% loss rate of requests, complete success.

Technology

  • Nodejs
  • Apache AB

Stack

  • 18 XLarge Amazon EC2 Instances
  • Amazon Elastic Load Balancing
  • Amazon RDS
  • NGINX

Process

Both the application and the requirements were very simple, bulk of time was spend using AB to support the benchmark.
