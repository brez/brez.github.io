BRESNIK

NYC DOT

Using a Wanco full matrix sign, developed a sequence based on a driver's speed (using supplied radar support) to trigger a specific sequence when a driver exceed the speed limit. The sign had only 3K wired memory to work with as well as being limited from flashing too fast by the Port Authority (within 5 miles of a port).

Technology

  • Wanco (WTMMB) Full Matrix
  • Wanco Simulator

Stack

  • FTP
  • Serial Port

Process

Used a Wanco simulator / worked with designer to prototype the sequence. Once complete, authored an installation manual and met with DOT personal in Queens to demonstrate how to install the sequence on an actual sign / test with a car.
